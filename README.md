# H-lipid-normalization

Here’s a quick note about the files:

“No_treatment” shows data from the untreated samples.
“Extracted” refers to the samples after the lipids were removed.
“Extracted_No_treatment” represents the difference between the extracted samples and the untreated samples.
“Lipid” contains data related to the lipid of the samples.
“Extracted_Lipid” represents the difference between the extracted samples and the lipid samples.
“New_data” is the validation dataset.
“Paatsjoki_2014_dH_water” contains environmental water data from Inari Lake.
